Install the open-iscsi prerequisite on all cluster nodes via a Longhorn-provided DaemonSet, managed by FluxCD. Longhorn's volume attachment mechanism requires `iscsid` to be active on every node — none of the three current nodes have it. This plan creates the GitOps manifests that install it automatically.

Purpose: open-iscsi must be running on all nodes BEFORE the Longhorn HelmRelease reconciles. Without iscsid, Longhorn manager pods start but volume attach/detach fails.

Output: FluxCD-managed DaemonSet that installs and enables open-iscsi on all nodes. Namespace and staging overlay wired into the existing controller hierarchy.
</objective>

**infrastructure/controllers/base/longhorn/iscsi-installer.yaml**
This is the official Longhorn iscsi-installer DaemonSet for v1.7.3. It uses `nsenter` to install `open-iscsi` on the host OS, enables `iscsid`, and loads the `iscsi_tcp` kernel module. The DaemonSet keeps a lightweight Alpine container running so it re-runs on any new node added to the cluster.

```yaml
# Source: https://raw.githubusercontent.com/longhorn/longhorn/v1.7.3/deploy/prerequisite/longhorn-iscsi-installation.yaml
# This DaemonSet installs open-iscsi on all cluster nodes before Longhorn HelmRelease reconciles.
# It uses nsenter to run commands in the host mount namespace (equivalent to running on the host).
# The init container does the actual installation; the main container is a no-op keep-alive.
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: longhorn-iscsi-installation
  namespace: longhorn-system
  annotations:
    command: &cmd apt-get install -y open-iscsi && systemctl enable iscsid && systemctl start iscsid && modprobe iscsi_tcp
spec:
  selector:
    matchLabels:
      app: longhorn-iscsi-installation
  template:
    metadata:
      labels:
        app: longhorn-iscsi-installation
    spec:
      hostNetwork: true
      hostPID: true
      initContainers:
        - name: iscsi-installation
          command:
            - nsenter
            - "--mount=/proc/1/ns/mnt"
            - "--"
            - bash
            - -c
            - *cmd
          image: alpine:3.17
          securityContext:
            privileged: true
      containers:
        - name: longhorn-iscsi-installation
          image: alpine:3.17
          command: ["/bin/sh", "-c", "while true; do sleep 86400; done"]
          securityContext:
            privileged: true
  updateStrategy:
    type: RollingUpdate
```
