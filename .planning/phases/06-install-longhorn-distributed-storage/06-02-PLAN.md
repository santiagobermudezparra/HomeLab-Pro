**infrastructure/controllers/base/longhorn/release.yaml**
Key decisions documented inline:
- `version: "1.7.3"` — pinned, not floating; re-verify if >30 days from research date 2026-04-05
- `install.crds: Create` + `upgrade.crds: CreateReplace` — required; chart ships 22 CRDs
- `persistence.defaultClass: true` — marks longhorn StorageClass as default
- `persistence.defaultClassReplicaCount: 2` — sets `numberOfReplicas: "2"` in the StorageClass parameters (used by K8s PVC provisioner)
- `defaultSettings.defaultReplicaCount: "2"` — sets Longhorn UI default for volumes (must be string "2", not integer)
- `ingress:` section intentionally omitted from this release — ingress is handled in Plan 03 as a standalone overlay file, consistent with the linkding pattern
- `metrics.serviceMonitor.additionalLabels.release: kube-prometheus-stack` — must match live cluster's `serviceMonitorSelector` label

```yaml
apiVersion: helm.toolkit.fluxcd.io/v2
kind: HelmRelease
metadata:
  name: longhorn
  namespace: longhorn-system
spec:
  interval: 30m
  chart:
    spec:
      chart: longhorn
      version: "1.7.3"
      sourceRef:
        kind: HelmRepository
        name: longhorn
        namespace: longhorn-system
      interval: 12h
  install:
    crds: Create
  upgrade:
    crds: CreateReplace
  values:
    persistence:
      defaultClass: true
      defaultClassReplicaCount: 2

    defaultSettings:
      defaultReplicaCount: "2"

    metrics:
      serviceMonitor:
        enabled: true
        additionalLabels:
          release: kube-prometheus-stack
```
