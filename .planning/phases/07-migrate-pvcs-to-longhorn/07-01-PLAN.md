<task type="auto">
  <name>Task 2: Live migration — backup pgadmin data, recreate PVC on Longhorn, restore data</name>
  <files></files>
  <read_first>
    - .planning/phases/07-migrate-pvcs-to-longhorn/07-RESEARCH.md (migration procedure and pitfall list, especially pitfall 1 about empty volume)

  </read_first>
  <action>
Execute the following sequence imperatively against the live cluster. Do NOT add any nodeSelector, nodeName, or nodeAffinity in the debug pod overrides — Kubernetes will schedule the debug pod on any available node, which is correct.

**Step 1 — Scale down pgadmin:**
```bash
kubectl scale deployment/pgadmin -n pgadmin --replicas=0
kubectl wait --for=delete pod -n pgadmin -l app=pgadmin --timeout=60s
```

**Step 2 — Spawn debug pod and copy data out:**
```bash
kubectl run debug-pgadmin \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"pgadmin-data-pvc"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/pgadmin-data"}]}]}}' \
  -n pgadmin
kubectl wait --for=condition=Ready pod/debug-pgadmin -n pgadmin --timeout=60s
mkdir -p /tmp/pgadmin-backup
kubectl cp pgadmin/debug-pgadmin:/pgadmin-data/. /tmp/pgadmin-backup/
kubectl delete pod/debug-pgadmin -n pgadmin --grace-period=0
```

**Step 3 — Delete old PVC and apply updated storage.yaml:**
```bash
kubectl delete pvc pgadmin-data-pvc -n pgadmin
kubectl apply -f apps/base/pgadmin/storage.yaml
kubectl wait pvc/pgadmin-data-pvc -n pgadmin \
  --for=jsonpath='{.status.phase}'=Bound --timeout=120s
```

**Step 4 — Restore data via new debug pod:**
```bash
kubectl run debug-pgadmin \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"pgadmin-data-pvc"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/pgadmin-data"}]}]}}' \
  -n pgadmin
kubectl wait --for=condition=Ready pod/debug-pgadmin -n pgadmin --timeout=60s
kubectl cp /tmp/pgadmin-backup/. pgadmin/debug-pgadmin:/pgadmin-data/
kubectl delete pod/debug-pgadmin -n pgadmin --grace-period=0
```

**Step 5 — Scale up and verify:**
```bash
kubectl scale deployment/pgadmin -n pgadmin --replicas=1
kubectl wait --for=condition=Ready pod -n pgadmin -l app=pgadmin --timeout=120s
```

**Pitfall guard (permission errors):** After scale-up, check `kubectl logs deployment/pgadmin -n pgadmin`. If "permission denied" on /var/lib/pgadmin, the debug pod restored with wrong ownership. Fix: `kubectl exec -n pgadmin deployment/pgadmin -- chown -R 5050:5050 /var/lib/pgadmin` — pgadmin runs as UID 5050.

**Cleanup:** `rm -rf /tmp/pgadmin-backup`
