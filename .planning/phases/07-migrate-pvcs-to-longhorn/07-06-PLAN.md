<task type="auto">
  <name>Task 2: Live migration — backup n8n-data PVC, recreate on Longhorn, restore data</name>
  <files></files>
  <read_first>

  <action>
Execute the following sequence imperatively against the live cluster. This migrates only the n8n-data PVC (app workflow data). The n8n-postgresql-cluster-1 PVC is handled in Plan 07 (CNPG). Do NOT add nodeSelector, nodeName, or nodeAffinity in the debug pod spec.

**Step 1 — Scale down n8n app:**
```bash
kubectl scale deployment/n8n -n n8n --replicas=0
kubectl wait --for=delete pod -n n8n -l app=n8n --timeout=60s
```

**Step 2 — Copy out n8n workflow data (1.1MB):**
```bash
kubectl run debug-n8n \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"n8n-data"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/n8n-data"}]}]}}' \
  -n n8n
kubectl wait --for=condition=Ready pod/debug-n8n -n n8n --timeout=60s
mkdir -p /tmp/n8n-backup
kubectl cp n8n/debug-n8n:/n8n-data/. /tmp/n8n-backup/
kubectl delete pod/debug-n8n -n n8n --grace-period=0
```

**Step 3 — Delete old PVC and apply updated storage.yaml:**
```bash
kubectl delete pvc n8n-data -n n8n
kubectl apply -f apps/base/n8n/storage.yaml
kubectl wait pvc/n8n-data -n n8n \
  --for=jsonpath='{.status.phase}'=Bound --timeout=120s
```

**Step 4 — Restore data via new debug pod:**
```bash
kubectl run debug-n8n \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"n8n-data"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/n8n-data"}]}]}}' \
  -n n8n
kubectl wait --for=condition=Ready pod/debug-n8n -n n8n --timeout=60s
kubectl cp /tmp/n8n-backup/. n8n/debug-n8n:/n8n-data/
kubectl delete pod/debug-n8n -n n8n --grace-period=0
```

**Step 5 — Scale up and verify:**
```bash
kubectl scale deployment/n8n -n n8n --replicas=1
kubectl wait --for=condition=Ready pod -n n8n -l app=n8n --timeout=120s
```

**Cleanup:** `rm -rf /tmp/n8n-backup`
