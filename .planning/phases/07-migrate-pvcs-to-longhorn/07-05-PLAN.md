<task type="auto">
  <name>Task 2: Live migration — backup linkding-data-pvc, recreate on Longhorn, restore data</name>
  <files></files>
  <read_first>

  <action>
Execute the following sequence imperatively against the live cluster. This migrates only the linkding-data-pvc (app bookmark data). The linkding-postgres-1 PVC is handled separately in Plan 07 (CNPG). Do NOT add nodeSelector, nodeName, or nodeAffinity in the debug pod spec.

**Step 1 — Scale down linkding app:**
```bash
kubectl scale deployment/linkding -n linkding --replicas=0
kubectl wait --for=delete pod -n linkding -l app=linkding --timeout=60s
```

**Step 2 — Copy out linkding app data (48KB SQLite database):**
```bash
kubectl run debug-linkding \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"linkding-data-pvc"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/ld-data"}]}]}}' \
  -n linkding
kubectl wait --for=condition=Ready pod/debug-linkding -n linkding --timeout=60s
mkdir -p /tmp/linkding-backup
kubectl cp linkding/debug-linkding:/ld-data/. /tmp/linkding-backup/
kubectl delete pod/debug-linkding -n linkding --grace-period=0
```

**Step 3 — Delete old PVC and apply updated storage.yaml:**
```bash
kubectl delete pvc linkding-data-pvc -n linkding
kubectl apply -f apps/base/linkding/storage.yaml
kubectl wait pvc/linkding-data-pvc -n linkding \
  --for=jsonpath='{.status.phase}'=Bound --timeout=120s
```

**Step 4 — Restore data via new debug pod:**
```bash
kubectl run debug-linkding \
  --image=busybox \
  --restart=Never \
  --overrides='{"spec":{"volumes":[{"name":"data","persistentVolumeClaim":{"claimName":"linkding-data-pvc"}}],"containers":[{"name":"debug","image":"busybox","command":["sleep","600"],"volumeMounts":[{"name":"data","mountPath":"/ld-data"}]}]}}' \
  -n linkding
kubectl wait --for=condition=Ready pod/debug-linkding -n linkding --timeout=60s
kubectl cp /tmp/linkding-backup/. linkding/debug-linkding:/ld-data/
kubectl delete pod/debug-linkding -n linkding --grace-period=0
```

**Step 5 — Scale up and verify:**
```bash
kubectl scale deployment/linkding -n linkding --replicas=1
kubectl wait --for=condition=Ready pod -n linkding -l app=linkding --timeout=120s
```

**Cleanup:** `rm -rf /tmp/linkding-backup`
